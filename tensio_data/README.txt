This folder contains trimmed field observations corresponding to individual irrigation events.

The file name format is 

tensio_[PLOT #]_[EVENT #]_[TENSIOMETER #]

Note that:
Plot 1 = March, High
Plot 2 = February, High
Plot 3 = March, Low
Plot 4 = February, Low
Plot 5 = Control

Events are listed in the events file in the upper directory.

Tensiometer numbers are defined in the tensiometer file. 

Numbering pattern with respect to depth:
([TENSIOMETER #] Mod 4) = 0 or 1 => 60cm placement
([TENSIOMETER #] Mod 4) = 2 or 3 => 150cm placement
